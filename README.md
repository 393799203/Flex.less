# Flex.Less

A [less] mixin for [flexbox].


## Available Mixins

The mixin names and their values correspond to the official draft.

* `display`
* `justify-content`
* `align-items`
* `align-self`
* `align-content`
* `order`
* `flex`
* `flex-grow`
* `flex-shrink`
* `flex-basis`
* `flex-direction`
* `flex-wrap`
* `flex-grow`
